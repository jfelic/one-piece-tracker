# One Piece Tracker

A watch progress tracker for the One Piece anime, built around the community-curated **Pace** guide — a watch order that removes filler episodes and trims pacing to save roughly 40% of total watch time.

## Features

- **Watch Tracker** — tracks your current episode and arc, with progress bars for both arc and overall completion
- **Arc Guide** — a full reference table of all 36 arcs with episode ranges, Pace episode counts, and time saved per arc
- Progress and current position are persisted in localStorage

## Stack

- [Next.js](https://nextjs.org) (Pages Router)
- [shadcn/ui](https://ui.shadcn.com) + Tailwind CSS
- Light / Dark / System theme support via `next-themes`

## Getting Started

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
